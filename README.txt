== Description ==

Official Mandoo™ plugin. This plugin allows you to integrate some features of your account Mandoo in your Wordpress admin panel for ease of use.

== Installation ==

1. Upload `mandoo-plugin.php` to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Go to Api Keys section an introduce your Mandoo API keys.

== Frequently Asked Questions ==

= ¿It's necessary a Mandoo Account to use the plugin? =

Yes. If yout don't have a Mandoo ID go to [Mandoo](https://users.mandoocms.com/signup/ "Create a account") and create a Mandoo Account.
